 # 🍽️🗑️ Food-Waste-Analytics-Patterns-Drivers-Operational-Efficiency
 A Power BI dashboard project analysing food waste patterns, identifying operational inefficiencies, and surfacing actionable insights across menu categories, 
 seasonal cycles, geographic locations, and supply chain drivers.

 ---

 ## 📂 Project Overview
 Food waste is one of the most pressing challenges in the food service industry — impacting profitability, sustainability, and operational efficiency. 
 This project uses Power BI to transform raw wastage records into an interactive analytical dashboard that helps food service operators and decision-makers:
  •	Understand what is being wasted and how much
  •	Identify when and where waste peaks occur
  •	Pinpoint the root drivers behind waste (pricing, storage, purchasing behaviour, geography)
  •	Measure operational efficiency using key performance indicators
  
 The dashboard is structured across two analytical pages: Food Waste Patterns and Food Waste Drivers, with a clean summary report view. 
 The dashboard answers critical questions like:
  •	What does waste look like, and where is it concentrated?
  •	Why is waste happening?
  •	Which food categories generate the most waste? 
  •	Which menu items have the highest waste percentage? 
  •	How does seasonality influence waste volumes? 
  •	What event types correlate with higher waste? 
  •	Does pricing strategy affect waste percentage? 
  •	How do storage conditions contribute to total waste? 
  •	Does purchase history/buying pattern predict waste levels? 
  •	Which geographical locations are the highest-waste zones?

 ## 🧹 Data Preparation Steps
 ☑️	Cleaned missing values and ambiguous perpetrator entries. 
 ☑️	Standardized call topics and agent names. 
 ☑️	Modelled relationships for efficient filtering. 
 ☑️	Built DAX measures for:
   •	Total waste
   •	Total prepared
   •	Total sold
   •	Efficiency percentage
   •	Sell-through percentage
   •	Waste percentage
